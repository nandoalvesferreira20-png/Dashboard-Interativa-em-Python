# 📊 Dashboard de Funcionários

Este projeto é um **dashboard interativo** feito com [Streamlit](https://streamlit.io/) e [Plotly](https://plotly.com/python/) para visualizar dados fictícios de funcionários.  
O dataset contém informações como **nome, idade, departamento, salário e cidade**.

---

## 🚀 Funcionalidades
- Filtros por **departamento** e **cidade**  
- Métricas rápidas:
  - Total de funcionários
  - Salário médio
  - Idade média
- Gráficos:
  - Salário médio por departamento
  - Distribuição de idades (faixa etária)
  - Top 10 cidades com mais funcionários
- Tabela interativa com os dados filtrados
